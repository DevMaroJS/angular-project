# Angular Project

This project is a simple Angular application built with the Angular CLI.

## Features

- Angular-based frontend structure
- Component-based architecture
- Basic home and housing location views
- Styled with CSS

## Prerequisites

Make sure you have the following installed:

- Node.js
- npm
- Angular CLI

## Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

## Run the application

Start the development server:

```bash
ng serve
```

Then open your browser and visit:

```text
http://localhost:4200/
```

## Build for production

```bash
ng build
```

## Project structure

```text
src/
  app/
    home/
    housing-location/
```
